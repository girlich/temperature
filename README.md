# temperature
Project to completely setup and manage a self-made temperature sensor system

Cabling:
* Power
  * Pin: 1
  * Column: Left
  * Row: 1
  * Usage: +3.3V
* Signal
  * Pin: 7
  * Column: Left
  * Row: 4
  * Usage: (GPCLK0) GPIO 4
* Ground
  * Pin: 30
  * Column: Right
  * Row: 15
  * Usage: Ground
